## 📌 Sobre
O *Jogo do Número Secreto* foi desenvolvido como parte das atividades do curso **Alura – Lógica de programação: mergulhe em programação com JavaScript**.
O objetivo do curso foi desenvolver o raciocínio lógico por meio da prática com JavaScript, utilizando desafios e pequenos projetos interativos. O foco principal não foi o aprofundamento em HTML e CSS, mas sim a aplicação de lógica de programação para construir funcionalidades dinâmicas.
Neste projeto, o usuário precisa adivinhar um número secreto gerado aleatoriamente pelo sistema dentro de um intervalo definido. Durante o jogo, o sistema fornece dicas indicando se o número secreto é maior ou menor que o valor informado, além de contabilizar o número de tentativas até que o jogador descubra a resposta.

## 🧠 Conceitos Fundamentais Praticados
Durante o desenvolvimento deste projeto, foram aplicados conceitos essenciais de programação, como:
- Estruturação e organização de variáveis
- Uso de condicionais para controle da lógica do jogo
- Criação e organização de funções
- Manipulação do DOM para atualizar elementos da interface
- Geração de números aleatórios
- Manipulação de arrays para controle de números sorteados
- Atualização dinâmica de mensagens na página
- Controle de estado da aplicação (tentativas e reinício do jogo)
