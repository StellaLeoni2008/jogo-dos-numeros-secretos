## 📌 Sobre

O *Jogo do Número Secreto* foi desenvolvido como parte das atividades do curso **Alura – Lógica de programação: mergulhe em programação com JavaScript**.

O objetivo do curso foi desenvolver o raciocínio lógico por meio da prática com JavaScript, utilizando desafios e pequenos projetos interativos.

Neste projeto, o usuário precisa adivinhar um número secreto gerado aleatoriamente pelo sistema dentro de um intervalo definido. Durante o jogo, o sistema fornece dicas indicando se o número secreto é maior ou menor que o valor informado, além de contabilizar a quantidade de tentativas realizadas até que o jogador descubra a resposta correta.

O foco principal do projeto foi aplicar lógica de programação para construção de regras, controle do fluxo da aplicação e manipulação dinâmica da interface.

---

## ⚙️ Funcionalidades
- Geração aleatória do número secreto
- Sistema de dicas para auxiliar o jogador
- Contagem de tentativas
- Reinício automático do jogo
- Atualização dinâmica das mensagens da interface

---

## 🧠 Conceitos Fundamentais Praticados
Durante o desenvolvimento deste projeto, foram aplicados conceitos essenciais de programação, como:
- Estruturação e organização de variáveis
- Uso de condicionais para controle da lógica do jogo
- Criação e reutilização de funções
- Manipulação do DOM para atualizar elementos da página
- Geração de números aleatórios
- Manipulação de arrays para controle de números sorteados
- Atualização dinâmica de mensagens na interface
- Controle do estado da aplicação
